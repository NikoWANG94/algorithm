### 题目：
斐波那契数，通常用 F(n) 表示，形成的序列称为斐波那契数列。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。也就是：

F(0) = 0,   F(1) = 1
F(N) = F(N - 1) + F(N - 2), 其中 N > 1.
给定 N，计算 F(N)。

```
示例 1：

输入：2
输出：1
解释：F(2) = F(1) + F(0) = 1 + 0 = 1.
```


### 解析1：
本质是DP，对DP进行了拆解。

* **复杂度：**
|  |复杂度|大小|百分比|
|--|--|--|--|
|时间|$O(n)$|36 ms|63.40%|
|空间|$O(1)$|13.1 MB|49.07%|

```python
class Solution:
    def fib(self, n: int) -> int:
        if n == 0:return 0
        f1,f2 = 0,1
        for i in range(n-1):
            f2,f1 = f1+f2,f2
        return f2 % 1000000007
```
