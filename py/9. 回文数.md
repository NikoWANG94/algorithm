### 题目：
判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。
```
示例 1:

输入: 121
输出: true

示例 2:

输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。
```

### 解析1：
转成字符串直接翻转，然后判断是否相等。

|  |复杂度|大小|百分比|
|--|--|--|--|
|时间|$O(n)$|72 ms|48.46%|
|空间|$O(n)$|11.6 MB|33.69%|


```python
class Solution(object):
    def isPalindrome(self, x):
        x = str(x)
        return x == x[::-1]
```
