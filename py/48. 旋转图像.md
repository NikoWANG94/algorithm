### 题目：

给定一个 n × n 的二维矩阵表示一个图像。

将图像顺时针旋转 90 度。

说明：

你必须在原地旋转图像，这意味着你需要直接修改输入的二维矩阵。请不要使用另一个矩阵来旋转图像。

```
示例 1:

给定 matrix = 
[
  [1,2,3],
  [4,5,6],
  [7,8,9]
],

原地旋转输入矩阵，使其变为:
[
  [7,4,1],
  [8,5,2],
  [9,6,3]
]
```

### 解析1：
找规律，我们发现，图像旋转，即图像矩阵经过了一次颠倒操作，然后转置。题目的问题是原地操作。

|  |复杂度|大小|百分比|
|--|--|--|--|
|时间|$O(n^2)$|52 ms|78.46%|
|空间|$O(1)$|13.8 MB|5.06%|

步骤：
1. 对图像矩阵进行颠倒；
2. 图形矩阵进行转置；

```python
class Solution:
    def rotate(self, matrix):
        matrix[:] = matrix[::-1]
        n = len(matrix[0])        
        # transpose matrix
        for i in range(n):
            for j in range(i, n):
                matrix[j][i], matrix[i][j] = matrix[i][j], matrix[j][i] 
```
